利用規約

以下はサーバーが提供するサービスにおける禁止事項や、サーバーコミュニティを保護するための利用規約(「本利用規約」といいます)です。
ユーザーは本ゲームサーバーに接続することにより、利用規約に従うことに同意したことになります。
本利用規約に同意できない場合には、本ゲームサーバーに接続したり、サービスを利用してはなりません。
本利用規約は事前のお知らせなしに変更できます。

1. 用語の定義

  • 「サーバー」とは、私たちのグループ Echore Server を意味します。
  • 「本ゲームサーバー」とは、私たちが提供する Minecraft: Bedrock Edition サーバーを意味します。
  • 「コミュニティーサーバー」とは、私たちがユーザーとのコミュニケーションを図るために運営、管理しているサーバーを意味します。(Discordに限られない)。

2. 禁止事項

以下に記載する禁止事項を破ることは、利用規約に違反することになります

  • 本ゲームサーバー、ウェブサイトなど私たちが提供するサービスに意図的に、または過度に負荷をかける行為
  • 意図していない動作の意図的な利用
  • 本ゲームサーバー内のチャット機能でサーバーが提供しているサービス以外の宣伝をする行為
  • サーバー内で提供しているサービスの例: パーティー、ゲームルームなど
  • 他者のアカウントや不正なアカウントを使用して本ゲームサーバーに接続する行為
  • 複数のアカウントを使用して処罰などを回避する行為
  • 有利に働く外部ツールまたはそれらに該当するものを使用する行為
    (マクロ、チートなど、通常プレイでできないことを可能にするツールは禁止です)
  • サーバーのサービス提供を妨害する行為
  • 権限者や開発者が不適切と判断する行為
  • 不適切な発言(ヘイトスピーチ、さらし、差別、偏見、暴言、下ネタなど他者が不快に感じるような発言)の投稿
  • 他ユーザーが不快に感じるような内容のメッセージの投稿
  • ウイルスなどのリンクやファイルの投稿
  • ウイルス: 端末に予期されない動作を起こす事を目的とした特異なプログラム
  • 運営に対しての過度な催促
  • 利用規約に違反しているユーザーを報告せずに黙認する行為
  • 有名な人物または運営に成りすまそうとする行為
  • 処罰を受けているにも関わらずそれを回避しようとする行為
  • 私たちのサービスに関連して、反社会的勢力に対して直接または間接に利益を供与する行為

3. サービス

  • 私たちはユーザーへの事前のお知らせなしにサービスの提供を停止できるものとします
  • 本サービスを利用したことにより発生したユーザーまたは第三者への不利益や損害について、私たちは理由を問わず一切の責任を負わないものとします
  • 本ゲームサーバー内でのアイテムや通貨の紛失、意図しない不利益について、理由を問わず私たちは補償する責任を持たないものとします

4. 処罰

私たちは本利用規約を違反したことのあるユーザーや、私たちが不適切と判断したユーザーに対して事前のお知らせなしに以下の処罰を行うことができます。
これらの処罰を行うかどうかは私たちの裁量に委ねられるものとします。

  • ユーザーデータの削除
  • 私たちの運営、管理するコミュニティサーバーからの追放
  • コミュニティーサーバーにおける投稿の削除
  • 本ゲームサーバーへの接続拒否
  • ユーザーが与えた本ゲームサーバー内への影響の無効化
  • 本ゲームサーバーでのチャット機能利用停止
  • コミュニティサーバー内でのメッセージ投稿制限

また処罰を行ったことにより発生したユーザーまたは第三者への不利益や損害について、私たちは理由を問わず一切の責任を負わないものとします

5. フォーラムとコミュニケーション

  • コミュニティサーバーにおいて、ユーザーが自由に意見を投稿したりできますが、私たちはその意見に同意したりするわけではなく、 意見や投稿内容について私たちは責任を負いません。
  • ユーザーはコミュニティサーバーにメッセージを投稿した場合、内容にかかわらず第三者の知的財産権やその他権利を侵害するものでないことに同意するものとします。
  • 私たちは、事前のお知らせなしにコミュニティサーバーの運営を放棄したり、コミュニティサーバーに投稿された内容を削除できる権利を保持します。
  • ユーザーはコミュニティサーバーにメッセージを投稿した場合、その内容がサーバーのサービスの一部としてコピーまたは改変して使用、配布されることに同意するものとします。

6. 連絡

  • 私たちはコミュニティサーバーを通じてユーザーに対して通知またはお知らせを行います。
  • お知らせを行う場合、私たちがユーザーに見えるメッセージを送信した時点で全ユーザーに対してお知らせを行ったとみなします。
